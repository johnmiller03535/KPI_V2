## ЧТО ДЕЛАТЬ

Добавить таб «KPI-карточки» в `/admin` (`frontend/src/app/admin/page.tsx`).

Таб состоит из двух секций:
1. **Библиотека показателей** — просмотр всех 126 показателей
2. **Карточка должности** — просмотр показателей конкретной должности

---

## СЕКЦИЯ 1 — БИБЛИОТЕКА ПОКАЗАТЕЛЕЙ

### Что показывать
Таблица показателей из `GET /api/kpi/indicators`:

| Колонка | Значение |
|---|---|
| Название | indicator.name |
| Тип | badge: binary_auto/binary_manual/threshold/multi_threshold/quarterly_threshold |
| Общий | бейдж «Общий» если is_common=true |
| Статус | badge: draft(warn)/active(ok)/archived(dim) |
| Кнопка | «Просмотр» |

Фильтры сверху:
- Dropdown «Все типы» / binary_auto / binary_manual / threshold / multi_threshold / quarterly_threshold
- Dropdown «Все статусы» / draft / active / archived
- Поиск по названию (client-side)

### Детали показателя (при клике «Просмотр»)
Модальное окно или раскрывающийся блок:
- Название, тип, статус
- Критерий оценки
- Для threshold/multi_threshold: числитель и знаменатель
- Пороги (thresholds) в читаемом виде: «>=67% → 100 баллов»
- Используется в N карточках

---

## СЕКЦИЯ 2 — КАРТОЧКА ДОЛЖНОСТИ

### Выбор должности
Dropdown сверху — все 91 должность.
При выборе загружать `GET /api/kpi/cards/{pos_id}/active`.

### Отображение карточки
Список показателей в карточке:
- Название показателя
- Тип (badge)
- Вес (%)
- Общий (badge если is_common)
- Сумма весов внизу с индикатором: зелёный если = 100%, красный если нет

### Дизайн
Dark Cyber стиль как везде в admin-панели.
