## ЗАДАЧА 1 — Сайдбар: только верхний уровень подразделений

Та же проблема что была в «Подчинённость» — сайдбар показывает все отделы
включая «Правовое управление. Отдел судебной практики» вместо просто «Правовое управление».

### Решение
Переиспользовать ту же функцию `normalizeUnit()` что уже написана для таба «Подчинённость»:

```typescript
function normalizeUnit(dept: string): string {
  return dept
    .split('.')[0]
    .replace(/\s+/g, ' ')
    .replace(/[-–]\s+/g, '-')
    .trim()
}
```

Вынести `normalizeUnit` в общий хелпер (например `utils/subordination.ts`)
и импортировать в обоих табах — не дублировать код.

Группировка и сортировка — та же логика:
- `Map<string, number>` для дедупликации
- «Руководство» первым, остальные по `localeCompare('ru')`
- Фильтрация правой части через `normalizeUnit(card.department).startsWith(selected)`

### Ожидаемый результат
```
📋 Все карточки          91
⚠️ Без карточки           0
─────────────────────────
Руководство               4
Правовое управление      12
Управление анализа...    11
...
```

## ЗАДАЧА 2 — Sticky-шапка и внутренний скролл

Та же проблема — при скролле пропадает заголовок «КАРТОЧКИ ДОЛЖНОСТЕЙ» и кнопка «+ Создать карточку».

### Решение
Тот же паттерн что уже реализован в «Подчинённость»:

```tsx
{/* Sticky шапка */}
<div style={{ position: 'sticky', top: 64px, zIndex: 30, background: 'var(--bg)' }}>
  <h2>КАРТОЧКИ ДОЛЖНОСТЕЙ</h2>
  <button>+ Создать карточку</button>
</div>

{/* Скроллящийся двухколоночный layout */}
<div style={{ display: 'flex', height: 'calc(100vh - 200px)', overflow: 'hidden' }}>
  {/* Левый сайдбар */}
  <div style={{ width: '340px', overflowY: 'auto', flexShrink: 0 }}>
    {/* список подразделений */}
  </div>
  {/* Правая часть */}
  <div style={{ flex: 1, overflowY: 'auto' }}>
    {/* список карточек или открытая карточка */}
  </div>
</div>
```
