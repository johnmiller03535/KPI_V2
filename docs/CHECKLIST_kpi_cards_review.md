# ЧЕКЛИСТ: Ручная проверка KPI-карточек в UI

> Дата: 2026-05-03 | Проходить после SQL-аудита

---

## КАК ПРОВЕРЯТЬ

1. Таб «KPI-карточки» → выбрать подразделение в левой панели
2. Открыть каждую карточку кнопкой «Открыть →»
3. Проверить по чеклисту ниже
4. Отметить ✅ или записать проблему

**Что проверять в каждой карточке:**
- [ ] Сумма весов = 100% (зелёный «✓ Верно» внизу)
- [ ] Все показатели правильного типа (Авто/Ручной/Порог/Мульти-порог/Квартальный)
- [ ] 3 общих показателя присутствуют (исполнительская дисциплина 30%, трудовой распорядок 10%, охрана труда 10%)
- [ ] Показатель «Просмотр» → критерий оценки заполнен (не пустой)
- [ ] Для threshold: числитель и знаменатель заполнены в «Просмотр»
- [ ] Для threshold: пороги отображаются (не пустые)

## ЧТО ДЕЛАТЬ С ПРОБЛЕМАМИ

**Сумма весов ≠ 100%** → «Редактировать» карточку, скорректировать веса

**Отсутствует показатель** → Проверить в «KPI-показатели» что показатель существует →
  добавить в карточку через «Редактировать» → «+ Добавить показатель»

**Пустой критерий/числитель** → «KPI-показатели» → «Редактировать» показатель → заполнить

**Пустые пороги** → «KPI-показатели» → «Редактировать» → добавить правила в визуальный редактор
