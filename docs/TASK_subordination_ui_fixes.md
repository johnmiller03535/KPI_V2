## ЗАДАЧА 1 — Левая панель: только верхний уровень подразделений

### Проблема
В режиме «По подразделениям» левая панель показывает все подразделения включая отделы:
```
Правовое управление                              ← нужно
Правовое управление. Отдел судебной практики     ← лишнее
Правовое управление. Отдел нормативного сопр.    ← лишнее
```

### Решение
Группировать по верхнему уровню — всё до первой точки.

**Frontend — `frontend/src/app/admin/page.tsx`:**

При построении списка подразделений для левой панели:
```typescript
// БЫЛО: уникальные значения department as-is
const departments = [...new Set(positions.map(p => p.department))]

// СТАЛО: брать только верхний уровень (до первой точки)
const topLevelDepts = [...new Set(
  positions.map(p => p.department?.split('.')[0]?.trim() ?? '')
)].filter(Boolean)
```

При фильтрации правой части — фильтровать по startsWith:
```typescript
// Клик на "Правовое управление" показывает всех у кого
// department.startsWith("Правовое управление")
const filtered = positions.filter(p =>
  p.department?.startsWith(selectedDept)
)
```

Счётчик рядом с подразделением — количество должностей у которых
`department.startsWith(deptName)`.

## ЗАДАЧА 2 — Sticky-заголовок таблицы при скролле

### Проблема
При прокрутке вниз заголовок таблицы (`ДОЛЖНОСТЬ / ПОДРАЗДЕЛЕНИЕ / РУКОВОДИТЕЛЬ / КАРТОЧКА`)
уходит за экран — непонятно какая колонка что значит.

### Решение

**Режим «Список»** — сделать строку заголовка sticky:
```css
/* Найти div/tr с заголовками колонок и добавить: */
position: sticky;
top: 0;         /* или top: 60px если есть sticky навбар выше */
z-index: 10;
background: #0a0a1a; /* var(--bg) или чуть светлее чтобы отделялось */
border-bottom: 1px solid rgba(0, 229, 255, 0.3);
```

**Режим «По подразделениям»** — правая часть тоже скроллится,
сделать её внутренний скролл с sticky-заголовком если он есть.
Если заголовков нет — достаточно зафиксировать левую панель
(`position: sticky; top: 0; height: 100vh; overflow-y: auto`).

### Важно
Проверить что sticky не перекрывает модалку «Изменить» (z-index модалки должен быть выше).
