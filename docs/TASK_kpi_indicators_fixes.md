## ЗАДАЧА 2 — Sticky-шапка и внутренний скролл

Тот же паттерн что уже реализован в табах «Подчинённость» и «KPI-карточки».

```tsx
{/* Sticky шапка */}
<div style={{ position: 'sticky', top: 64, zIndex: 30, background: 'var(--bg)' }}>
  <div style={{ display: 'flex', gap: 12 }}>
    <input placeholder="Поиск по названию..." />
    <button>+ Добавить показатель</button>
  </div>
</div>

{/* Двухколоночный layout с внутренним скроллом */}
<div style={{ display: 'flex', height: 'calc(100vh - 200px)', overflow: 'hidden' }}>
  {/* Левый сайдбар */}
  <div style={{ width: '280px', overflowY: 'auto', flexShrink: 0 }}>
    {/* фильтры по группам */}
  </div>
  {/* Правая часть */}
  <div style={{ flex: 1, overflowY: 'auto' }}>
    {/* список показателей */}
  </div>
</div>
```

Использовать уже готовые хелперы из `frontend/src/utils/admin.ts` где применимо.
