# ЗАДАЧА: Sticky-заголовок в режиме «По подразделениям»

> Файл для Claude Code | Дата: 2026-05-02 | Приоритет: 🟡

---

## Проблема

В режиме «По подразделениям» при скролле заголовок «МАТРИЦА ПОДЧИНЕНИЯ»
и кнопки переключателя уходят за экран. В режиме «Список» это работает
корректно — нужно то же поведение.

## Решение

Обернуть весь таб в layout с фиксированной высотой:

```
┌─ sticky: заголовок + кнопки ──────────────────────────┐
│ МАТРИЦА ПОДЧИНЕНИЯ     [Список] [По подразделениям] [Импорт] │
└────────────────────────────────────────────────────────┘
┌─ overflow: auto (скроллится) ─────────────────────────┐
│ ┌─ левая панель ─┐  ┌─ правая часть ────────────────┐ │
│ │ Все должности  │  │ Должности подразделения        │ │
│ │ Руководство    │  │ ...                            │ │
│ │ ...            │  │                                │ │
│ └────────────────┘  └────────────────────────────────┘ │
└────────────────────────────────────────────────────────┘
```

**Структура:**
```tsx
{/* Этот блок sticky */}
<div style={{ position: 'sticky', top: 0, zIndex: 20, background: 'var(--bg)' }}>
  <h2>МАТРИЦА ПОДЧИНЕНИЯ</h2>
  <div>{/* переключатель + кнопка импорта */}</div>
</div>

{/* Этот блок скроллится */}
<div style={{ display: 'flex', overflow: 'hidden', height: 'calc(100vh - 200px)' }}>
  {/* Левая панель */}
  <div style={{ overflowY: 'auto', width: '340px', flexShrink: 0 }}>
    {/* список подразделений */}
  </div>
  {/* Правая часть */}
  <div style={{ overflowY: 'auto', flex: 1 }}>
    {/* список должностей */}
  </div>
</div>
```

Это то же самое что уже сделано для режима «Список» — применить
тот же паттерн к двухколоночному layout.
