## СПРАВКА — ТИПЫ KPI

| Тип | Кто оценивает | Как |
|---|---|---|
| `binary_auto` | AI (YandexGPT) | Анализирует трудозатраты из Redmine |
| `binary_manual` | Руководитель | Кнопки ✅/❌ в форме проверки |
| `threshold` / `multi_threshold` | Автоматически | Сотрудник вводит факт, система считает score |
| `quarterly_threshold` | Автоматически | То же, но пороги зависят от квартала |
